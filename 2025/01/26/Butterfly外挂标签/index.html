前言

  1. Markdown外挂标签格式是Hexo独有格式，并不适用于其他地方
  2. 以下写法只适用于Butterfly主题，用在其他不会有效果

选项卡标签
  1. 选项卡标签参数（写在tabs后面）
    1. name:
      • 选项卡标签的名称
      • 必选
    2. [index]:
      • 默认分页索引
      • 默认值为1
      • 如果输入-1,则默认不选择分页
      • 可选
        2.分页标签参数（写在tab后面）
    3. Tab name:
      • 分页标签标题
      • 未指定则使用选项卡标签标题后最
      • 指定了图标，但未指定标题，则使用图标
      • 可选
    4. [@icon]:
      • icon图标名称
      • 会在标题前面添加一个图标
      • 可选

  4. 大小

  2. icon
    • 可配置自定义icon(只支持fontawesome图标)
    • 可选
  3. style
    • simple, modern, flat, disabled
    • 可选

  3. checked
