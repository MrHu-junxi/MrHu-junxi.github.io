MarkDown语法

Markdown 语法是一种轻量级标记语言，旨在用简洁的符号为文本添加结构化格式，使其易读易写，同时可轻松转换为 HTML 等格式。在许多博客或者网站文章发布方面，都会使用MarkDown语法。

标题

一级最大，六级最小

段落与换行

  • 段落：文本连续输入即可，段落间用空行分隔。
  • 换行：句尾加两个或多个空格，再按回车键，或用 HTML中的 <br> 标签。

行内文本样式

样式 语法 效果
加粗 **文本**或者__文本__ 加粗文本
斜体 *斜体文本*或_斜体文本_ 斜体文本
斜体加粗 ***斜体加粗文本***或___斜体加粗文本___ 斜体加粗文本
删除线 ~~删除线文本~~ 删除线文本
下划线 <u>下划线文本</u> 下划线文本
